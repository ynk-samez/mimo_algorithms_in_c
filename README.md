## ダウンロード 
### gitをインストール済みの場合
自分のPCのコマンドラインで
```bash
git clone  https://github.com/ynk-samez/mimoSimlations.git
```
### gitをインストールしてない場合
  ダウンロードボタンからzipをダウンロードして回答

## 実行方法
- ex) 最尤推定法(MLD法を実行する場合)はコマンドラインで
```bash
cd MLD
make 
```
- ex) 最尤推定法(ZF法を実行する場合)はコマンドラインで
```bash
cd ZF
make 
```
出力結果はResultsフォルダに出力される.
